# websnap

```
$ ./websnap.js -h

  Usage: websnap [options] <output>

  Options:

  -h, --help           output usage information
  -V, --version        output the version number
  -u, --url <url>      url
  -o, --output <path>  image output path
```

## example

```
./websnap.js -u https://geta6.booth.pm -o booth_capture_geta6.png
```
